提出一种在水下自动获取自由游泳状态鱼类的鱼体长度、宽度和重量的方法，该方法基于单阶段目标检测新算法，使用了ZED第二代3D感测相机，对水下移动目标的正交视图进行快速、准确识别并定位，获取正交姿态锚框信息。
然后使用ZED双目相机获取左右视图，并根据左右视图计算出深度图，通过水下双目相机估计的深度和实际测量深度拟合深度模型，使用鱼体长宽数据和重量数据拟合鱼体重量估计模型。
然后使用YOLOv5得到的锚框，深度模型和相机参数计算真实鱼体长宽。最后，使用真实长宽和鱼体重量估计模型估计鱼体体重。主要应用于水产养殖鱼类生物量无接触式全自动化实时智能估算。
基于python实现的yolov5算法，ZED的SDK，以及pyqt5等库实现，能够准确、快速地估计鱼类在自由游泳状态下的长度、宽度和重量，进而提高鱼类生物量无损智能测量的效率与精度。
包括基于pyqt5的主界面，yolov5算法相关模块，图像输入方式选择模块，IOU,置信度选择模块，ZED双目图像获取和深度获取模块，图像预处理模块，水下深度估计模块，鱼体长宽估测模块，鱼体重量估测模块，鱼体生物量统计模块，图像和估计值保存模块，图像实时显示模块。
